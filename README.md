# Litetest

Litetest is a lite app that allows you to make a ~2s speedtest, and doesn't collect any of your data.

If you have a bug, please open an [Issue](https://github.com/Nb1X/Litetest/issues).

If you have a good idea for the app and you have the code, please send an e-mail: internet_zzb10x@ik.me with your code.
